# Phishi-Finder

2023학년도 1학기 숭실대학교 네트워크 프로그래밍 네트워크 패킷 활용 자유주제 프로젝트

## 개요

이 프로젝트는 URL이 유사한 피싱 사이트를 네트워크 패킷을 분석하여 검출하는 시스템을 개발하는 것을 목표로 합니다.  
이를 통해 접속한 페이지의 잠재적인 피싱 공격을 식별하고 사용자를 보호하며, 보다 확실한 피싱 페이지 탐지 솔루션을 제공합니다.
